# Lindo Hogar Online (LHO) - Store Repository

This repository contains the source code, theme configurations, and assets for the Lindo Hogar Online (LHO) e-commerce storefront.

## Tech Stack
- Platform: Shopify (Proof of Concept)
- Payments: Wompi / MercadoPago
- Logistics: Envía / Coordinadora
